# MLOPS-Git_Practice

This repository is a practice project for learning the basics of Git version control in the context of MLOps (Machine Learning Operations). It demonstrates fundamental Git concepts such as commits, branches, and conflict resolution through simple Python scripts.

## Project Structure

- `Demo.py`: A Python script that demonstrates Git commits, branching, and merge conflicts.
- `dummy.py`: A simple dummy Python file for additional Git practice.
- `LICENSE`: MIT License file.
- `README.md`: This file.

## Getting Started

### Prerequisites

- Python 3.x installed on your system.

### Running the Scripts

To run the demo script:

```bash
python Demo.py
```

To run the dummy script:

```bash
python dummy.py
```

## Git Practice

This project is designed to practice the following Git concepts:

- Making commits
- Creating and switching branches
- Merging branches
- Resolving merge conflicts
